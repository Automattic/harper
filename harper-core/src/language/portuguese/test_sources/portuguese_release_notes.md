# Notas de versão do produto

## Resumo

A nova versão inicia mais rápido e responde de forma mais estável com listas longas.
Um servidor de banco de dados adicional distribui picos de carga mais cedo e mantém a interface estável mesmo sob alta carga.

## Melhorias

A correção de erro na dialog de importação evita entradas duplicadas.
Os requisitos do sistema permanecem inalterados, mas o espaço em disco é utilizado de forma mais eficiente.
Uma nota de versão curta explica as principais alterações para a equipe de suporte.
