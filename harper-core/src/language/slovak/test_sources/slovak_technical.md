# Technická dokumentácia

## Systémové požiadavky

Pre inštaláciu potrebujete:
- Operčný systém Linux alebo Windows
- Aspoň 4 GB RAM
- 500 MB priestoru na disku

## Konfigurácia

Konfiguračný súbor sa nachádza v priečinku `/etc/harper`.
Môžete upraviť nastavenia pomocou textového editora.

## Odstraňovanie chýb

Ak sa program nespustí, skontrolujte prosím:
1. Systémové požiadavky
2. Inštaláciu
3. Konfiguráciu

## Bežné problémy

Tu sú niektoré bežné problémy a ich riešenia:
- Program sa nespustí: Skontrolujte inštaláciu
- Konfigurácia sa nenahrá: Skontrolujte oprávnenia súboru
- Výkon je pomalý: Zväčšte operačnú pamäť
