# Správa o verzii produktu

## Zhrnutie

Nová verzia sa spúšťa rýchlejšie a reaguje stabilnejšie pri dlhých zoznamoch.
Dodatočný databázový server distribuuje špičky zaťaženia skôr a udržuje rozhranie stabilné aj pri vysokom zaťažení.

## Vylepšenia

Oprava chyby v importnom dialógu zabraňuje duplicitným záznamom.
Systémové požiadavky ostávajú nezmenené, ale priestor na disku sa využíva efektívnejšie.
Krátka informácia o verzii vysvetľuje najdôležitejšie zmeny pre tím podpory.
